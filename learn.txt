Virtual Environment:

python3 -m venv myenv → create
source myenv/bin/activate → activate (Mac/Linux)
myenv\Scripts\Activate → activate (Windows)
deactivate → exit


Best Practices:
Keep a requirements.txt file so others can install the same dependencies:
pip freeze > requirements.txt

Later, install all at once:
pip install -r requirements.txt

Usually, add myenv/ to .gitignore so you don’t push your virtual environment to GitHub.
